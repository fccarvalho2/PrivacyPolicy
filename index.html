1. Introdução

Obrigado por usar o MIND.Funga! Sua privacidade é importante para nós, e esta Política de Privacidade explica como coletamos, usamos e protegemos seus dados ao utilizar nosso aplicativo. Ao acessar ou usar o aplicativo, você concorda com os termos descritos nesta política.

2. Informações que Coletamos

Ao utilizar o aplicativo MIND.Funga, coletamos os seguintes dados:

  • Dados de Localização: O aplicativo coleta a localização GPS do seu dispositivo apenas quando você envia uma imagem para análise.
  • Dados de Envio: Registramos a data e o horário em que você envia uma imagem.
  • Dados de Registro: Durante o registro, coletamos:
    • Nome
    • Sobrenome
    • Endereço de e-mail
    • Universidade/Instituição (opcional, para candidatos a curador)
    • Área de especialidade (opcional, para candidatos a curador)

3. Finalidade da Coleta de Dados

Usamos os dados coletados para:

  • Processar e analisar imagens enviadas usando Redes Neurais Convolucionais (CNNs) de Aprendizado Profundo.
  • Fornecer resultados de identificação de espécies e informações relacionadas.
  • Melhorar a precisão e funcionalidade do aplicativo.
  • Contribuir para pesquisas mapeando e monitorando a distribuição de espécies de macrofungos ao longo do tempo.
  • Facilitar a colaboração com curadores e especialistas para expandir o banco de dados.

4. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos seus dados pessoais com terceiros para fins de marketing. No entanto, podemos compartilhar dados anonimizados com pesquisadores e colaboradores para apoiar estudos científicos e melhorar os modelos de identificação.

5. Armazenamento e Segurança de Dados

Estamos comprometidos em manter seus dados seguros:

  • Todos os dados são armazenados com segurança em servidores gerenciados por [Inserir Provedor/Instituição].
  • O acesso aos dados é restrito apenas a pessoal autorizado.
  • Protocolos de criptografia e segurança protegem os dados durante a transmissão e armazenamento.

6. Direitos do Usuário

Você tem o direito de:

  • Acessar e revisar seus dados.
  • Solicitar correções ou atualizações em suas informações.
  • Excluir sua conta e os dados associados mediante solicitação.
  • Retirar o consentimento para coleta de dados opcionais (por exemplo, universidade/instituição e área de especialidade).

7. Retenção de Dados

Mantemos seus dados pelo tempo necessário para fornecer nossos serviços, conduzir pesquisas e cumprir obrigações legais. Mediante solicitação, seus dados podem ser excluídos permanentemente.

8. Privacidade de Crianças

O aplicativo é destinado a usuários com 13 anos ou mais. Não coletamos intencionalmente dados de crianças menores de 13 anos. Pais ou responsáveis podem entrar em contato conosco para solicitar a exclusão de dados de menores.

9. Alterações nesta Política de Privacidade

Podemos atualizar esta política periodicamente. Quaisquer alterações serão publicadas nesta página e notificaremos os usuários registrados por e-mail.
